# 🧮 Calculator App

A **responsive Calculator** built using **HTML, CSS, and JavaScript**, capable of performing all basic arithmetic operations.  
The UI is clean and intuitive, making it easy to perform calculations on desktop or mobile devices.

---

## 🚀 Features

- ➕ Addition  
- ➖ Subtraction  
- ✖️ Multiplication  
- ➗ Division  
- % Percentage calculation  
- ⌫ Delete last input  
- 🧹 Clear all input  
- 📱 Responsive and user-friendly interface  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3 (Flexbox for layout)  
- JavaScript (DOM manipulation and event handling)  

---

## 💡 What I Learned

- Manipulating the **DOM** to update display dynamically  
- Handling **events** for button clicks  
- Implementing **basic arithmetic logic** in JavaScript  
- Building **responsive layouts** using CSS Flexbox
